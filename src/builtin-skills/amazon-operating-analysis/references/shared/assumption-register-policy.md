# 假设登记规则

## 必备字段

| 字段 | 用途 |
|---|---|
| 假设编号 | 便于后续追问和复核的稳定编号。 |
| 变量 | 业务变量或资料项。 |
| 取值 | 当前取值或场景范围。 |
| 来源 | 用户提供、安信默认、模型默认、AI 暂估、外部规则。 |
| 资料等级 | M0、M1、M2 或 M3。 |
| 置信度 | 高、中、低。 |
| 结论影响 | 该假设变化会影响哪个结论。 |
| 披露话术 | 在顾问或客户输出中如何披露。 |

## 来源规则

- 用户提供：说明文件、字段、期间或用户原话。
- 安信默认：说明默认值、适用范围和确认人。
- 模型默认：说明来自财务模型原型。
- AI 暂估：只用于草稿探索，不能支撑正式测算。
- 外部规则：说明来源日期；规则可能变化时必须核验。

## 禁止沉默假设

税费、认证合规成本、账期、平台冻结款、长期仓储费、季节性、促销计划、断货影响、平台政策费用和固定费用分摊，不得沉默假设。
