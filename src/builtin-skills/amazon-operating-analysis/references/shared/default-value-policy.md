# 默认值政策

## 默认值类型

| 类型 | 使用条件 | 披露方式 |
|---|---|---|
| 安信默认 | 安信确认过的经营规则或经验值。 | 写明值、范围和来源。 |
| 模型默认 | 来自 `跨境电商创业财务模型_v2.xlsx`。 | 写明来自模型，仍需确认。 |
| 用户计划假设 | 用户目标或计划值。 | 标记为用户计划，不是历史证据。 |
| AI 暂估 | 只用于探索草稿。 | 结论最高只能到暂估测算。 |

## 可谨慎默认

支付手续费、普通平台月费和低影响杂费可以在披露后暂用默认值。

汇率不得自行估算汇率。用户未提供汇率时，必须先调用 `exchange_rate_lookup` 查询市场参考汇率，或要求用户/财务人员确认。市场参考汇率不能自动等同于财务记账汇率；用于正式核算、结算、记账或老板复核口径时，必须取得财务人员确认。

## 不建议默认

税费、认证合规、平台冻结款、采购账期、物流付款节点、长期仓储、断货影响、季节性和平台政策费用不得沉默默认。

## 输出规则

任何默认值只要进入计算，都必须先进入假设表，再经过输出门禁。
