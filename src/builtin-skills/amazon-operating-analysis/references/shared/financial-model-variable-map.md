# 财务模型变量映射

参考原型：`docs/requirements/assets/跨境电商创业财务模型_v2.xlsx`。

该工作簿只用于识别变量和公式框架，不是运行时模型。里面的默认值必须经安信确认后才能作为业务默认。

## 变量组

| 变量组 | 变量 | 使用方 |
|---|---|---|
| 售价与需求 | 售价、稳定月销量、12 个月销量爬坡 | 单件模型、立项 BP、未来预测 |
| 商品成本 | 采购成本、国内运费、头程、重量 | 单件模型、成本结构 |
| Amazon 变动费用 | FBA 配送费、FBA 仓储费、销售佣金、支付手续费 | 单件模型、成本结构 |
| 广告与退货 | ACoS、广告订单占比、退货率、退货损失率 | 单件模型、立项 BP、未来预测、敏感性 |
| 固定与启动成本 | 平台月费、人员、办公、社保、一次性启动费 | 成本结构、立项 BP、现金流 |
| 现金输入 | 初始资金、采购付款时点、平台回款时点 | 立项 BP、现金流预测 |
| 敏感性变量 | 售价、销量、ACoS、退货率 | 敏感性与盈亏平衡 |
| 政策驱动变量 | duty_rate、vat_rate、platform_fee_rate、fba_fee、compliance_cost、certification_cost、customs_clearance_cost、lead_time、logistics_restriction、subsidy_amount | 成本结构、单件模型、敏感性、现金流、库存风险、立项 BP |

## 公式原型

| 输出 | 原型关系 | 注意 |
|---|---|---|
| 单件毛利 | 售价减商品、FBA、平台、广告、退货等成本 | 税费和合规成本可能缺失。 |
| 月度经营利润 | 单件利润乘销量，再减固定费用 | 不代表现金一定够用。 |
| 12 个月 BP | 月度销量爬坡加成本假设 | 只属于场景规划。 |
| 盈亏平衡 | 固定成本除以贡献毛利 | 必须先完成成本分类。 |
| 可承受 ACoS | 毛利空间折算为广告成本上限 | 必须披露广告销售口径。 |
| 政策影响测算 | 政策驱动变量变化 -> 成本、售价、利润、现金流或库存约束变化 | 影响变量来自 `policy_impact_handoff`，正式测算前必须补齐资料。 |

## 使用规则

本文件用于统一变量命名。需要公式细节或模板假设时，再检查原始工作簿。
