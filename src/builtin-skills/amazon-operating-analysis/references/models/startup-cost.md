# 一次性启动费用规则

## 来源

吸收自 Excel `一次性启动费用` 工作表。最低值、最高值、中间值均保留；默认测算使用中间值，但必须披露为模型默认假设。

## 费用全集

| 类别 | 费用项 |
|---|---|
| 注册与资质 | 营业执照注册、对公银行账户、美国商标、中国商标、产品认证、UPC/EAN、VAT税号、进出口权。 |
| 品牌与设计 | 品牌 VI、产品包装设计、产品摄影、A+/EBC、Listing 多语言、品牌旗舰店、Shopify 建站、社交媒体素材。 |
| 首批产品采购 | 样品采购与测试、首批库存采购、第三方质检、包装材料、模具/开模、产品改良。 |
| 首批物流费用 | 国内运输、头程海运、头程空运、报关报检、关税与进口税、海外仓/FBA 入库、货运保险。 |
| 工具与软件 | Amazon 专业卖家账户、ERP、选品关键词工具、邮件营销、设计工具、财务记账、收款工具、客服工具、数据分析工具。 |

## 计算规则

| 输出 | 公式 | 说明 |
|---|---|---|
| 单项中间值 | `(最低预估 + 最高预估) / 2` | 四舍五入到元。 |
| 类别合计 | 同类别单项求和 | 输出最低、最高、中间值。 |
| 启动费用合计 | 五大类求和 | 默认用于立项 BP 启动投入。 |
| 美元等值 | CNY / 汇率 | 仅用于展示，不改变本位币口径。 |

## 门禁

- 首批库存采购、认证、商标、VAT、关税等不得沉默忽略。
- 如果客户明确不做 Shopify、欧洲站、开模、空运等，应在假设表标记为排除项。
- 启动费用是现金投入，不等同利润费用；是否摊销进经营利润必须单独说明。
