# 12 个月销量爬坡规则

## 基准爬坡

| 月份 | 稳定销量比例 | Excel 说明 |
|---|---:|---|
| M1 | 0% | 筹备期，无销售。 |
| M2 | 25% | 上架初期，少量出单。 |
| M3 | 50% | 广告启动，销量爬升。 |
| M4 | 65% | 持续优化，稳步增长。 |
| M5 | 80% | 评价积累，转化提升。 |
| M6 | 100% | 达到稳定销量。 |
| M7 | 115% | 超越基准，持续增长。 |
| M8 | 125% | 旺季/口碑效应。 |
| M9 | 130% | 稳定增长。 |
| M10 | 140% | 品类扩展效应。 |
| M11 | 150% | 规模效应显现。 |
| M12 | 165% | 年末旺季冲刺。 |

## 使用规则

- 该曲线是立项 BP 的默认基准场景，不是市场预测事实。
- 用户没有历史基线时，只能叫“计划测算”或“场景测算”。
- 如果产品有明显季节性、断货、上新、促销、大促节奏，必须替换或拆分该曲线。
- 现金流可以复用该曲线，但需要叠加回款和付款时点。
