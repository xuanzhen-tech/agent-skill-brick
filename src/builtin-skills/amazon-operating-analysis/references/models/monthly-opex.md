# 月度运营费用规则

## 来源

吸收自 Excel `月度运营费用` 工作表。该表用于稳定期成本结构和保本分析，不直接代表现金付款时点。

## 费用全集

| 类别 | 性质 | 费用项 |
|---|---|---|
| 平台固定 | 固定 | Amazon 专业卖家月费、Shopify 月费、域名续费、SSL 证书。 |
| 佣金扣费 | 变动 | Amazon 销售佣金、FBA 配送费、FBA 仓储费、收款手续费、汇损预留。 |
| 采购补货 | 变动 | 产品采购成本、头程物流补货、包装耗材、质检抽检费。 |
| 广告营销 | 变动/预算 | Amazon PPC、站外社媒广告、KOL、优惠券、Vine、品牌推广/DSP。 |
| 人员办公 | 固定 | 运营团队工资、办公场地杂费、社保公积金、培训、差旅。 |
| 售后合规 | 混合 | 退货处理费、退货商品损失、库存损耗报废、VAT/GST、财税代理、产品责任保险。 |

## 标准公式

- 月销售额 = 稳定月销量 * 售价 USD * 汇率。
- 费用占销售额 = 费用 / 月销售额。
- 固定费用 = 平台固定 + 人员办公 + 售后合规中固定项。
- 变动费用 = 佣金扣费 + 采购补货 + 广告营销 + 售后合规中随销售变化项。
- 汇损预留沿用 Excel 原型 1%，但必须披露。
- VAT/GST 沿用 Excel 原型 3%，正式测算需按站点和税务口径确认。

## 差异

Excel `占月销售额` 公式存在括号问题，标准口径采用 `费用 / (销量 * 售价 * 汇率)`。
