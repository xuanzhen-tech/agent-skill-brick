# 财务 BP 概念与指标口径

## 定位

本资料把 `跨境电商创业财务模型_v2.xlsx` 中的表格概念抽象为 Agent 可复用的财务 BP 语言。Agent 应使用这些概念理解问题，确定性计算交给 `packages/tools/src/anxin_tools/operating_analysis`。

## 核心对象

| 对象 | 业务含义 | 一期边界 |
|---|---|---|
| 新品/新项目 | 尚未形成稳定历史经营事实的 Amazon 产品或项目。 | 做经济可行性和风险测算，不替代选品决策。 |
| 单件经济模型 | 每卖 1 件商品时，售价被采购、物流、平台、广告、退货等变动项侵蚀后的贡献空间。 | 不含人员办公、一次性启动费和完整税务判断。 |
| 启动费用 | 项目开始前或早期一次性现金投入。 | 用于启动资金和回本，不直接计入单件贡献利润。 |
| 月度运营费用 | 稳定期每月发生的固定和变动经营费用。 | 用于经营利润、成本结构和保本，不等于现金付款时点。 |
| 销量爬坡 | 从筹备期到稳定期、增长期的 12 个月销量假设。 | 是场景假设，不是市场事实。 |
| 现金流 | 现金实际流入、流出和期末余额。 | 必须区分收入确认、平台回款、银行到账、采购付款。 |
| 敏感性 | 在售价、销量、ACoS、退货率变化时检验利润边界。 | 不能提升基础模型结论等级。 |
| 盈亏平衡 | 固定成本和单件贡献利润共同决定的保本销量/销售额。 | 仅用于测算边界，不是销售承诺。 |

## 指标命名

| 推荐名称 | 不推荐名称 | 定义 |
|---|---|---|
| 单件贡献利润 | 单件净利润 | 售价 - 单件变动成本；不含固定费用和一次性费用。 |
| 广告后单件贡献毛利 | 广告后净利 | 单件贡献利润已扣 PPC 广告、退货和支付手续费。 |
| 月度经营利润 | 月净利润 | 单件贡献利润 * 销量 - 月度固定费用 - 当期摊销。 |
| 现金净流入 | 月利润 | 当期现金流入 - 当期现金流出。 |
| 期末现金余额 | 账面利润 | 上期现金 + 当期现金净流入。 |
| 启动资金需求 | 回本本金 | 一次性费用、首批采购、物流、首月费用和安全垫。 |
| 回本周期 | 资金安全 | 累计现金回收或累计经营利润覆盖初始投入所需月份，必须说明口径。 |

## 关键关系

- 售价同时影响销售收入、佣金、广告费、退货损失、支付手续费和利润率。
- 销量同时影响采购、头程、FBA、广告、退货、补货现金流和库存风险。
- ACoS 必须说明口径；Excel 原型使用 `售价 * ACoS * 广告出单占比`。
- 启动费用影响启动资金和回本，不应沉默摊入单件利润。
- 库存占用会解释“利润为正但现金紧”，但库存事实应先由库存资料或 ledger 生成。
- 税费、认证、合规、账期、平台冻结款、长期仓储费不得沉默默认。
