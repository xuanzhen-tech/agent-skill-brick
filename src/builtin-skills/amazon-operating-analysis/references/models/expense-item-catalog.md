# 费用项目录

## 口径

费用必须先按明细项计算，再汇总为类别。Agent 不得只解释大类金额；涉及成本结构、资金占用、回本和保本时，必须能追溯到明细项。

## 一次性启动费用

| 类别 | 费用项 | 性质 |
|---|---|---|
| 注册与资质 | 营业执照注册、对公银行账户开户、美国商标注册、中国商标注册、产品认证、UPC/EAN条形码、VAT税号注册、进出口权办理 | 一次性现金支出 |
| 品牌与设计 | 品牌VI设计、产品包装设计、产品摄影、A+/EBC页面设计、Listing文案撰写、品牌旗舰店搭建、独立站建站、社交媒体素材制作 | 一次性现金支出 |
| 首批产品采购 | 样品采购与测试、首批库存采购、产品质检费、包装材料与辅料、模具/开模费、产品改良/微创新费用 | 一次性现金支出 |
| 首批物流费用 | 国内运输至港口/机场、头程海运费、头程空运费、报关报检费、关税与进口税、海外仓入库费/FBA入库费、货运保险 | 一次性现金支出 |
| 工具与软件 | 亚马逊专业卖家账户、ERP管理系统、选品与关键词工具、邮件营销工具、设计工具、财务/记账软件、收款工具开户、客服工具、数据分析工具 | 一次性现金支出 |

## 月度运营费用

| 类别 | 明细项 | 性质 |
|---|---|---|
| 平台固定 | 亚马逊专业卖家月费、Shopify月费、域名续费、SSL证书 | 固定现金支出 |
| 佣金扣费 | 亚马逊销售佣金、FBA配送费、FBA仓储费、收款工具手续费、汇损预留 | 变动现金支出 |
| 采购补货 | 产品采购成本、头程物流补货、包装耗材、质检抽检费 | 变动现金支出 |
| 广告营销 | 亚马逊PPC站内广告、站外社媒广告、网红/KOL合作、促销折扣/优惠券、Vine/早期评论计划、品牌推广/DSP广告 | 预算/变动现金支出 |
| 人员办公 | 运营团队工资、办公场地+杂费、社保公积金、培训/学习费用、差旅费 | 固定现金支出 |
| 售后合规 | 退货处理费、退货商品损失、库存损耗/报废、VAT/GST税费、财务/税务代理费、产品责任保险 | 变动或混合现金支出 |

## 单件经济费用

| 项目 | 性质 |
|---|---|
| 产品售价 | 收入，不是费用 |
| 亚马逊销售佣金、FBA配送费、FBA仓储费、产品采购成本、国内运输、头程国际物流、PPC广告费、退货损失、收款手续费、其他杂费 | 单件变动费用 |
| 单件贡献利润 | 售价扣除单件变动费用后的贡献，不等同完整净利润 |

## 现金流费用

现金流按月保留销售收入、实际回款、一次性启动费用、产品采购、头程物流、平台佣金+月费、FBA配送+仓储费、广告推广、人员+办公+社保、其他运营费用。现金流支出必须说明付款时点，不能用利润口径替代现金口径。
