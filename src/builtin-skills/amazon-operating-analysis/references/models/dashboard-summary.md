# 总览仪表盘与输出摘要规则

## 来源

吸收自 Excel `总览仪表盘`。该表的价值不是导航 UI，而是定义立项 BP 的第一屏结论语言。

## KPI

| KPI | 标准口径 |
|---|---|
| 产品售价 | USD 售价和 CNY 等值。 |
| 单件贡献利润 | 标准单件贡献利润，不使用参数面板漏项公式。 |
| 贡献利润率 | 单件贡献利润 / 售价。 |
| 月度稳定销量 | 参数面板稳定销量或用户确认销量。 |
| 月度贡献利润 | 单件贡献利润 * 月度稳定销量。 |
| 年化贡献利润 | 月度贡献利润 * 12。 |
| 一次性启动费用 | 启动费用中间值或用户预算。 |
| 月度运营费用合计 | 稳定期月度运营费用。 |
| 回本周期 | 必须说明是资金池回收、项目投入回收还是现金回正。 |

## 关键洞察模板

- 所有默认数据必须说明来自模型默认或用户输入。
- 最大成本项应来自计算器的成本结构排序，而不是固定写死。
- ACoS 是高影响变量，必须提示敏感性。
- 12 个月现金流用于识别早期资金压力，不能只看单件利润。
- 预算有限时可提示“减少非必要启动项”的方向，但不替代经营决策。
- 建议保留安全现金垫时，必须说明这是管理建议边界。
